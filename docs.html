Purpose and Scope

This document provides a high-level overview of the Tax Consultant Application, a conversational AI system built using Chainlit that integrates multiple external services through Model Context Protocol (MCP) servers. The application enables users to interact with an AI agent that can perform tax-related calculations, access file systems, communicate via Slack, process payments through Stripe, and handle multilingual conversations across 13+ languages.

For detailed architecture information, see Architecture. For configuration specifics, see Configuration. For internationalization details, see Internationalization. For tax domain knowledge, see Data and Domain Knowledge.

System Overview

The Tax Consultant Application is a Chainlit-based web application that provides conversational AI capabilities for tax consulting. The system uses a LangChain ReAct agent powered by OpenAI's GPT-4o model, with access to multiple external services through MCP (Model Context Protocol) tool integration.

Core Capabilities

The Tax Consultant Application provides the following key capabilities:

Conversational AI Interface

  • Chainlit Web UI: Modern chat interface with streaming responses
  • Multi-language Support: 13+ supported languages including English, Hindi, Japanese, Chinese, Arabic, and European languages
  • Session Management: Conversation history and context preservation

Integrated Tool Access

  • File System Operations: Read, write, and manage local files through MCP filesystem server
  • Slack Integration: Send messages and interact with Slack workspaces
  • Payment Processing: Handle transactions through Stripe integration
  • Tax Data Processing: Extract and analyze tax information from documents

AI Agent Capabilities

  • ReAct Framework: Reasoning and acting loop for complex task completion
  • Tool Selection: Intelligent selection of appropriate tools based on user queries
  • Context Awareness: Maintains conversation context and user preferences
